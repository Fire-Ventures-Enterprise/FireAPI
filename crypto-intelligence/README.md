# 🔥 Crypto Intelligence API

Advanced cryptocurrency prediction and analysis service with multi-timeframe AI predictions.

## Features
- Multi-timeframe predictions (1h, 6h, 24h, 48h)
- Real-time price data scraping
- Technical analysis (RSI, MACD, Bollinger Bands)
- Confidence scoring and risk assessment
- Web interface for tracking predictions

## Endpoints
- `/predict/<coin>/<timeframe>` - Get predictions
- `/trending/<timeframe>` - Top opportunities
- `/track` - Prediction accuracy tracking
