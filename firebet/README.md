# FireBet - Sports Betting Intelligence Service

FireBet is a comprehensive sports betting intelligence service that provides real-time data, predictions, and analytics for sports betting decisions. Built as part of the Fire Ventures Enterprise ecosystem.

## 🔥 Features

- **Multi-Sport Support**: Basketball, Soccer, Baseball, American Football, Hockey
- **Real-time Data**: Live odds, scores, and game updates
- **Prediction Engine**: AI-powered betting predictions with contextual factors
- **Ordered Data Collection**: Consistent prediction factors (referee, weather, venue, travel, etc.)
- **Real-time Sync**: WebSocket integration for live updates
- **Analytics Dashboard**: Performance tracking and insights
- **Fire API Integration**: Seamless integration with Fire Ventures Enterprise ecosystem
